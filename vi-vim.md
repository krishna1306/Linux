### Command Mode

```bash
# copy
yy

# paste
p

# save
ZZ

# delete a char
x

# delete a line
dd

# delete 3 lines
d3d

# undo
u

# redo
r
```

#### Search

```bash
# search downwards
/line

# Find next - n
# Find previous - N
```
